- Skala Interval
Skala interval adalah skala yang digunakan untuk tidak hanya untuk mengklasifikasikan maupun 
memberikan tingkatan pada suatu titik pengamatan, 
namun kita dapat mengukur seberapa besar nilai antara suatu titik pengamatan dengan titik pengamatan lainnya. 
Beberapa contoh yang termasuk skala interval diantaranya adalah suhu tubuh dan jarak.

- Skala Rasio
Skala rasio memiliki kemiripan dengan skala interval, perbedaannya terletak pada nilai 0 pada skala rasio. 
Berbeda dengan skala interval yang tidak memiliki nilai 0 yang tidak pasti.
